# demo-multi-module

spring boot multi module

* demo-db
* demo-core
* demo-api
* demo-web

# How to run

### Build code

##### Maven build

```bash
cd [target repository dir]
mvn clean install -Dmaven.test.skip -Dcheckstyle.skip
```

# Test URL
- [demo-web](http://localhost:8080/)
- [demo-api](http://localhost:8081/)
